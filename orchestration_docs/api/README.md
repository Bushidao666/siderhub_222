# API (HTTP) — Escopo do subsistema de agents

Este diretório é reservado para endpoints REST do produto. O subsistema de orquestração `.agents` não define endpoints HTTP próprios — ele opera via arquivos JSONL e processos locais.

Consulte os serviços e interfaces em:
- `../services` — daemons e loops de orquestração
- `../interfaces` — contratos dos arquivos JSONL/estado
