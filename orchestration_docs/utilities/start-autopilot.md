# Utility: start-autopilot.sh

## Localização
`.agents/bin/start-autopilot.sh`

## Propósito
Bootstrap de sessão Codex para `subagent-main-autopilot`, capturando `session_id` e gravando PID.

## Saídas
- `.agents/sessions/subagent-main-autopilot.json`
- `.agents/pids/autopilot.pid`
- `.agents/logs/autopilot-bootstrap.log`
